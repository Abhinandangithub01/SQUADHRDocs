# 📋 Attendance Module

## 🧾 Introduction

The **Attendance module** helps organizations track employee presence, punctuality, and leave in a **single centralized system**. It simplifies daily attendance capture, automates **policy-based calculations**, and provides **real-time visibility** for **HR, managers, and employees**.

## 📊 Features Overview

| Feature                     | Description                                                                 |
|---------------------------|-----------------------------------------------------------------------------|
| 🕒 In/Out Tracking        | Record employee check-in and check-out times                                |
| 📝 Regularization         | Correct missing or incorrect attendance entries                             |
| 🌴 Leave Management       | Apply and approve leave requests                                            |
| ⏰ Attendance Monitoring  | Track late arrivals, early exits, overtime, and absences                    |
| 📊 Reports & Dashboards   | View attendance insights and trends                                         |
